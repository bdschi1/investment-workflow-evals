# Golden Answer — Returns Bridge / Value-Creation Decomposition

Sponsor equity check at entry = $330M (including $30M management
rollover). Exit equity value = $1,115M. Hold period = 4.5 years.

Total MOIC = $1,115M / $330M = **3.38x**. XIRR over 4.5 years ≈
**30.8%**.

## Decomposition Methodology

Start from entry equity of $330M. Walk through each driver, holding
others constant, to exit equity of $1,115M. Order matters only for
cross-term allocation; the total always ties.

### 1. EBITDA Growth (at entry multiple)

Hypothetical equity if EBITDA moved from $80M to $130M but multiple
stayed at 9.5x and debt stayed at $400M:

```
Hypothetical EV = $130M × 9.5 = $1,235M
Hypothetical equity = $1,235M − $400M = $835M
Contribution vs entry = $835M − $330M = +$505M
```

**EBITDA-growth bucket: +$505M** (48.8% of $1,035M total gain
ex-rollover effects).

### 2. Multiple Expansion (at exit EBITDA)

Going from 9.5x to 10.5x at exit EBITDA of $130M:

```
Incremental EV = $130M × (10.5 − 9.5) = $130M
Multiple-expansion contribution = +$130M
```

**Multiple-expansion bucket: +$130M** (12.6%).

### 3. Debt Paydown

Net debt fell from $400M to $250M:

```
Debt-paydown contribution = $400M − $250M = +$150M to equity
```

**Debt-paydown bucket: +$150M** (14.5%).

### 4. Cross-Term (Reconciliation)

Sum so far: $330M + $505M + $130M + $150M = $1,115M. Exactly matches
exit equity, so cross-term is **$0** in this illustrative case.
In cases with timing effects or non-linear interactions, a
reconciliation row surfaces them; here it is zero by construction.

### Total Bridge

| Step | Equity ($M) | $/sh equivalent |
|---|---:|---|
| Entry sponsor equity | 330 | baseline |
| + EBITDA growth | +505 | largest lever |
| + Multiple expansion | +130 | market-driven |
| + Debt paydown | +150 | capital-structure |
| + Cross-term | 0 | — |
| **Exit equity** | **1,115** | — |

## MOIC/IRR at Each Step

| Step | Cumulative Equity | MOIC | IRR (over 4.5 yrs) |
|---|---:|---:|---:|
| Entry | 330 | 1.00x | — |
| + EBITDA growth | 835 | 2.53x | ~22.7% |
| + Multiple expansion | 965 | 2.92x | ~26.5% |
| + Debt paydown | 1,115 | 3.38x | ~30.8% |

## Narrative for LPs (probabilistic, 3 sentences)

Roughly 49% of dollar value creation came from EBITDA growth at the
entry multiple — this is the operational piece that the sponsor
influenced directly. About 13% came from multiple expansion (a
benchmark-market tailwind rather than sponsor-specific action), and
roughly 15% from deleveraging through the hold. Realized returns
reflect both operational execution and a constructive exit market;
if exit multiples had returned to entry levels, MOIC would have been
closer to 3.0x and IRR meaningfully lower, which is worth acknowledging
rather than claiming the returns as entirely operational.
