# Qualification Module

This module contains qualification assessments for new contributor onboarding.

## Purpose

The qualification assessment:

1. Verifies domain knowledge and analytical skills
2. Demonstrates understanding of platform quality standards
3. Establishes baseline quality metrics for tier assignment
4. Familiarizes contributors with task formats and expectations

## Assessment

### Standard Qualification (30-45 minutes)
- Basic equity analysis scenario
- Tests fundamental analytical skills
- Required for all new contributors

## Scoring

| Score | Result |
|-------|--------|
| 85%+ | Fast-track to Standard tier |
| 70-84% | Pass - assigned to Trainee tier |
| 50-69% | Conditional pass - additional training required |
| <50% | Not passed - can retry in 7 days |
